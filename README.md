# Java-Assignment-1
A console-based shopping cart application built in Java as part of RMIT University's  Java Programming course (COSC3109). 
The program allows users to:
- Create items to purchase with name, price and quantity
- Add items to a shopping cart
- View cart contents and total cost
- Modify items in the cart
- Checkout and clear the cart

Built using core Object-Oriented Programming principles including encapsulation, 
constructors, getters/setters, and arrays. No ArrayList or HashMap used.

Technologies: Java
